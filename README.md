# Getting Started

## Clone Repository :

```bash
git clone https://github.com/aaridhon19/Test-FE-Kabar-Haji-Indonesia-.git
```

## Install Depedencies:

```bash
npm install
# or
yarn install
```

## Running Client:

```bash
npm run dev
```
The client will run on http://localhost:PORT
